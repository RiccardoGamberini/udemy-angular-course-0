Settings list page.

'settings' is the value of this button routerLink attribute. Angular tries to load http//localhost:4201/settings/list/settings. This will cause an error.

'./settings' is the value of this button routerLink attribute. Angular tries to load http//localhost:4201/settings/list/settings. This will cause an error.

'../settings' is the value of this button routerLink attribute. Angular tries to load http//localhost:4201/settings/settings. This will cause an error.

'settings' is the value of this button routerLink attribute. Angular tries to load http//localhost:4201/settings and this page will successfully loaded. Adding double dots and slash works the same as navigating in nested directories using windows PowerShell but with URL segments. I will explain what is an URL segment with an example. Example: the string 'settings/list' is composed by the segments 'settings' and 'list'. '../something' will remove one segment and will add 'something'. '../../something' will remove two segments and will add 'something'.
